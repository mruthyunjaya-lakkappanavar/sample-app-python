# Contributing

Thank you for your interest in this project.

## Contribution Policy

**This repository only accepts contributions from explicitly added collaborators.**

- External pull requests from non-collaborators will be **closed without review**.
- If you would like to contribute, please [open an issue](../../issues/new) first to discuss the proposed change.
- Only after being added as a collaborator by the repository owner will your contributions be considered.

## How to Request Contributor Access

1. Open an issue describing what you'd like to contribute.
2. Wait for the repository owner (@mruthyunjaya-lakkappanavar) to review and respond.
3. If approved, you will be added as a collaborator and can then submit pull requests.

## Rules for Collaborators

- All changes must go through a pull request (no direct pushes to `main`).
- All pull requests require approval from the repository owner before merging.
- Follow existing code style and conventions.
- Include appropriate tests for any changes.
- Do not remove or modify the LICENSE or copyright notices.

## Code of Conduct

Be respectful and constructive in all interactions.
